# Kisan Unnati Frontend

A Next.js application for empowering farmers with AI-driven insights.

## Features

- **Dashboard**: Centralized hub for all farmer tools
- **Weather**: Real-time weather information and forecasts
- **Crop Advice**: AI-powered crop recommendations
- **Schemes**: Government schemes and subsidies information
- **Marketplace**: Buy and sell agricultural products
- **Community**: Connect with other farmers
- **Rewards**: Earn points and redeem rewards
- **Voice Assistant**: Voice-controlled interactions
- **Multi-language Support**: English, Hindi, Telugu, Tamil

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/`: Next.js app router pages
- `components/`: Reusable React components
- `lib/`: Utility functions and API wrappers
- `styles/`: Global styles and CSS
- `public/`: Static assets

## Technologies Used

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Axios for API calls
- Speech Recognition API
- Speech Synthesis API
